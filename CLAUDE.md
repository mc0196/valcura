# ValCura

Demo cliccabile di una piattaforma di assistenza di prossimità per anziani in valle — prototipo-pitch per i soci fondatori, dati finti, nessun backend. Vedi `README.md`.

## Agent skills

### Issue tracker

Issues live in GitHub Issues on `mc0196/valcura` (via the `gh` CLI); external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage labels with default names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
